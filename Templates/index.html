ABOUT

The website will aim that users can get to know the idea from which disease he/she may be suffering. We will develop a website for taking inputs and give the predicted disease. We have the dataset by which we will predict the disease from the symptom's user is observing in him/her. For that prediction we will apply a few machine learning models and pick the best fitted algorithm according to accuracy. Anyone can do it by sitting at their home and can get the list of doctors to whom they can consult. For this doctors list, we will do web scraping by beautiful soap. This project will make it easy for the user to consult the doctors of specific diseases otherwise users have to move from one doctor to another.In this website i.e. Symptomatic Assistance Website. This project involves both Machine Learning and Web Development Technologies. The project shows the prediction of the diseases which will be predicted by the symptoms entered by the user. The data used to predict the disease contains 133 symptoms and 41 diseases. To predict the appropriate disease, first, we will clean the data, then pre-process the data, do some analysis, train the model and apply that algorithm which gives the best accuracy for the machine learning model.
